# Water Bottle

Vacuum-insulated sport bottle inspired by the supplied technical cutaway drawing (24oz bottle with threaded lid and loop cap).

## What changed

- Reworked proportions toward a 74mm-class body + shoulder + neck stack.
- Added translucent outer shell and separate internal cavity so interior contents are visible.
- Added explicit **water volume** with a slight meniscus dome for clearer liquid rendering.
- Added neck thread collar, taller cap, and a carry-loop feature to better match the reference silhouette.

## Parameters

- Body Radius / Height
- Shoulder Height
- Neck Radius / Height
- Wall and Bottom Thickness
- Water Fill %
- Cap Height
