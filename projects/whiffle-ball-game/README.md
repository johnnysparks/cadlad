# Whiffle Ball Carnival Game

Tabletop carnival game where players toss whiffle balls into scoring pockets.

## Design Intent

A carnival-style toss game with a tilted backboard containing circular pockets of varying sizes and point values. Larger pockets are easier to land but worth fewer points. The board has a raised rim to catch misses and a front lip for ball return.

## Key Dimensions

| Parameter | Default | Range | Notes |
|---|---|---|---|
| Board Width | 300mm | 200-500mm | Overall width |
| Board Height | 400mm | 300-600mm | Overall height |
| Board Tilt | 15deg | 5-30deg | Lean-back angle |
| Pocket Count | 5 | 3-9 | Number of scoring pockets |
